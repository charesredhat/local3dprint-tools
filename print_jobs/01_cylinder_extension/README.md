# 01 — Cylinder Extension

Hollow sleeve that fits **snugly over a 15.4 mm OD cylinder**.

| Parameter | Value |
|-----------|-------|
| Inner diameter | 15.6 mm (0.2 mm clearance) |
| Outer diameter | 19.6 mm |
| Wall thickness | 2.0 mm |
| Height | 25.0 mm |
| Perimeters | 5 × 0.4 mm nozzle |
| Layer height | 0.2 mm |
| Material | PLA |
| Est. filament | ~1138 mm (~3.4 g) |

## Files
- `generate_stl.py` — generates `cylinder_extension.stl`
- `generate_gcode.py` — generates `cylinder_extension.gcode`
- `cylinder_extension.stl` — mesh file
- `cylinder_extension.gcode` — ready-to-print (uploaded to OctoPrint)
