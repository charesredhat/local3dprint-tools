## How to Use

### First-time Setup

```bash
cd /mnt/1tssd-a/projects2/3dprinting

# 1. Copy the example config and fill in your values
cp .env.example .env
# Edit .env — set OCTOPRINT_URL and OCTOPRINT_API_KEY

# 2. Install dependencies
pip3 install requests python-dotenv numpy-stl
```

### Run the Manager

```bash
python3 octoprint_manager.py
```

**Menu options:**

| Option | Action |
| --- | --- |
| 1 | Show current printer status, temps, and job progress |
| 2 | List all files/folders on OctoPrint |
| 3 | Start the cylinder_extension print (job 01) |
| 4 | Upload any local `.gcode` file to a remote folder |
| 5 | Cancel the current print |
| 6 | Monitor current print (polls every 30 s) |
| 7 | Show local print_jobs folder tree |
| q | Quit |

---

### Adding a New Print Job

1. Create a folder: `print_jobs/02_your_part/`
2. Place your `.gcode` file inside it
3. Run the manager → option **4** to upload
   - Local path: `print_jobs/02_your_part/your_file.gcode`
   - Remote folder: `print_jobs/02_your_part`

---

### Regenerate the Cylinder STL or GCode

```bash
cd print_jobs/01_cylinder_extension
python3 generate_stl.py       # outputs cylinder_extension.stl
python3 generate_gcode.py     # outputs cylinder_extension.gcode
python3 generate_4x_stl.py    # outputs cylinder_extension_4x.stl
python3 generate_4x_gcode.py  # outputs cylinder_extension_4x.gcode
```

Edit the constants at the top of each file to change dimensions, height, wall thickness, etc.

---

### Direct API Access (curl examples)

> Replace `$OCTOPRINT_URL` and `$OCTOPRINT_API_KEY` with values from your `.env`.

```bash
# Printer status
curl -H "X-Api-Key: $OCTOPRINT_API_KEY" $OCTOPRINT_URL/api/printer

# List files
curl -H "X-Api-Key: $OCTOPRINT_API_KEY" $OCTOPRINT_URL/api/files

# Start a specific file
curl -X POST \
  -H "X-Api-Key: $OCTOPRINT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"command":"select","print":true}' \
  $OCTOPRINT_URL/api/files/local/print_jobs/01_cylinder_extension/cylinder_extension.gcode

# Cancel print
curl -X POST \
  -H "X-Api-Key: $OCTOPRINT_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"command":"cancel"}' \
  $OCTOPRINT_URL/api/job
```
